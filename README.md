1. Class Mahasiswa
Atribut:
nim → untuk menyimpan Nomor Induk Mahasiswa
nama → untuk menyimpan nama mahasiswa

Constructor:
Constructor default mengisi NIM dan Nama bawaan.
Constructor berparameter untuk mengisi data sesuai input.

Method:
setNim() dan setNama() → mengubah data.
getNim() dan getNama() → mengambil data.
tampilkanData() → menampilkan informasi mahasiswa.


2. Class MainClaas
Berisi main() sebagai pusat menjalankan program.
Menggunakan:
Scanner → membaca input dari pengguna.
ArrayList<Mahasiswa> → menyimpan banyak objek mahasiswa.
Program memakai menu (tambah data, tampilkan data, keluar).
Setiap data baru disimpan sebagai objek Mahasiswa dan dimasukkan ke ArrayList.
